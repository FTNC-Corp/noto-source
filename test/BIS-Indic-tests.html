It is expected that these links and web pages will be explored and investigated by the native speakers/readers who will check how correct and readable are the characters or sample texts on the pages available here.

The fonts used by this browser to display these pages are the default fonts on the phone or the device you are using. If this is a standard Android device which wasn't modified by a phone manufacturer then this non-Latin font belongs to the Noto Sans family (see get/noto for more information).

Font rendering issues could have multiple symptoms:

  • If you see a rectangle ("tofu") instead of a proper character then it means that the font on your device doesn't support a character required for this language. This situation should be noted (please make sure that you save the text that caused it somewhere) and brought up to the attention of the phone/device manufacturer.
  • A character is not recognizable as presented by the browser on some of the web pages. This situation should be noted (please make sure that you save the text that caused it somewhere) and brought up to the attention of the phone/device manufacturer.

The table below has the following columns and related test possibilities:
  • Language: identifies the language. If there's a wikipedia entry in a variant of this language then you'll see a link there. Please check if the web page is readable that you see is readable. Please make a note of any text that's not readable.
  • ICA Requirements: is a set of characters identified by the Indian Cellular Association (ICA) as required to support a given language by a given script. Any "tofu" there needs to be reported. Same if any of the characters are incorrect.
  • GSUB Tests: is a limited set of tests using character combinations that compose (SUBstitue to) a new character. Any "tofu" there needs to be reported. Same if any of the characters are incorrect.
  • UDHR or Other Sample: UDHR stands for Universal Declaration of Human Rights. It is published by the Office of the United Nations High Commissioner for Human Rights (our copy comes from the UNICODE organization). UDHR is easily accessible sample text available for a language and script(font) combination. Please check the readability.
